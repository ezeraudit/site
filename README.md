# 弈泽财报分析（Ezer Audit）

弈泽（Ezer）是一套相对固定、可复核、可追踪的财报分析工作流。

**网站地址： https://audit.ezer.cc/ （English Version: https://audit.ezer.cc/ ）**

## 💡 为什么发起这个项目？

在当前的投资环境下，普通投资者面临的痛点不是“结论太少”，而是“逻辑黑盒”。

- 信息过载：每天都有海量的复盘、研报和情绪判断。
- 立场偏差：传统卖方研报天然服务于机构与产业，其观点往往是市场结构的一部分，而非中立答案。
- 黑盒结论：很多分析直接给出一个目标价或评级，但缺乏一套能把结论拆开检查的标准化流程。

**Ezer Audit 想做的，是把那些优秀投资者分析里反复出现的关键步骤，尽量标准化、结构化、流程化。**

## 🎯 核心价值

- 利益透明：我们的长期利益来自工具效率的提升，而非具体股票的涨跌或市场情绪的煽动。
- 逻辑可复核：不只是给结论，而是展示从原始数据到最终判断的推导过程。
- AI 协同：利用 AI 重做财报分析流程，处理结构化数据、拆解指标、识别风险，最后由人工进行复核。

## 📊 分析维度

弈泽 并不创造新的理论，而是将资深投资者的经验转化为一套稳定的分析 SOP：

1. 利润质量：剔除一次性因素，看清真正的盈利能力。
2. 现金流匹配：验证利润的含金量，检查真金白银是否到账。
3. 增长持续性：拆解增长动能，判断是行业红利还是内生增长。
4. 资产负债压力：识别隐藏在报表深处的财务隐患。
5. 估值与预期：厘清市场目前是在交易“确定性”还是“想象力”。
